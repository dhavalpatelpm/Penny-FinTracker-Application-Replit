# 💰 Penny – AI Powered Personal Finance Tracker

**Penny** is a modern **AI-powered personal finance tracking application** that helps users monitor their **income, expenses, and savings** with intelligent insights and a clean dashboard.

Instead of wondering *“Where did my money go?”*, Penny provides **clear financial visibility and smart spending insights.**

# 🏗️ System Architecture

Penny follows a **modern full-stack architecture** combining a responsive frontend, backend APIs, database storage, and AI-powered financial insights.

```
                ┌─────────────────────┐
                │        User         │
                └─────────┬───────────┘
                          │
                          ▼
                ┌─────────────────────┐
                │      Frontend       │
                │   (React / Web UI)  │
                └─────────┬───────────┘
                          │ API Requests
                          ▼
                ┌─────────────────────┐
                │       Backend       │
                │   (Node.js / API)   │
                └───────┬─────┬───────┘
                        │     │
                        │     │
                        ▼     ▼
            ┌──────────────┐  ┌────────────────┐
            │   Database   │  │   AI Engine    │
            │ Transactions │  │ (Gemini / LLM) │
            │ User Data    │  │ Insights       │
            └──────────────┘  └────────────────┘
```

# ⚙️ Architecture Components

## 1️⃣ Frontend Layer

Responsible for **user interaction and visualization**.

**Key Responsibilities**

* Display financial dashboard
* Manage expense and income entries
* Show transaction history
* Visualize financial insights

**Technologies**

* React
* Responsive UI
* API integration

---

## 2️⃣ Backend Layer

Handles **application logic and API management**.

**Responsibilities**

* Process financial transactions
* Handle user data
* Generate analytics
* Integrate AI-based financial insights

**Technology**

* Node.js
* REST APIs

---

## 3️⃣ Database Layer

Stores structured financial information.

**Data Stored**

* User profiles
* Income records
* Expense transactions
* Financial summaries

**Possible Databases**

* PostgreSQL
* MongoDB
* Firebase / Supabase

---

## 4️⃣ AI Insights Layer

AI analyzes financial behavior and provides **intelligent recommendations**.

**Capabilities**

* Spending pattern analysis
* Financial behavior insights
* Smart savings suggestions
* AI-powered recommendations

**AI Models**

* Google Gemini
* LLM-based financial assistant

---

# 🔄 Data Flow

1️⃣ User adds a transaction
2️⃣ Frontend sends request to backend API
3️⃣ Backend processes and validates the request
4️⃣ Transaction is stored in the database
5️⃣ AI engine analyzes financial data
6️⃣ Dashboard updates analytics and insights
