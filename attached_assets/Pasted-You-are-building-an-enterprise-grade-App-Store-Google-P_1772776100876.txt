Mobile:
- React Native + Expo SDK 51 (managed workflow)
- TypeScript (strict mode, no 'any')
- Expo Router v3 (file-based routing: tabs + stacks + modals)
- React Native Reanimated 3 (all animations)
- React Native Gesture Handler (swipe, drag interactions)
- Victory Native XL (charts: pie, donut, bar, line, area)
- React Native Linear Gradient (liquid glass + backgrounds)
- Expo Blur (BlurView for glass morphism effect)
- React Native MMKV (ultra-fast local storage, persisted state)
- Zustand (global state management)
- React Hook Form + Zod (form validation)
- React Native Date Picker (Expo-compatible)
- Expo Print + Expo Sharing (PDF generation + share sheet)
- React Native SVG (custom icons and illustrations)
- Expo Haptics (haptic feedback on all interactions)
- Expo Font (custom fonts)
- React Native Safe Area Context (safe areas on all devices)
- React Native Keyboard Avoiding View (form UX)
- Lottie React Native (animated illustrations for empty states/onboarding)
- date-fns (all date formatting and calculations)
- Lucide React Native (base icon set)

Backend (Lightweight, Scalable):
- Node.js + Express + TypeScript
- PostgreSQL via Supabase (auth + database + realtime)
- Supabase JS Client (from mobile app directly)
- Prisma ORM (schema management)
- Anthropic Claude API claude-sonnet-4-20250514 (AI suggestions page)
- Row Level Security (RLS) on all Supabase tables
- Supabase Auth (email/password + magic link)

Architecture Pattern:
- Offline-first: all data cached in MMKV, synced to Supabase when online
- Optimistic UI updates (instant feedback, background sync)
- Repository pattern for data access layer
- Shared TypeScript types across app
- Feature-based folder structure (not type-based)
